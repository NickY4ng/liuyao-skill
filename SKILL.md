description: 六爻占卜 Skill，支持交互式起卦和直接输入两种模式。触发词：起卦、算一卦、摇卦、🎲、六爻、占卦。

## 使用方式

### 1. 交互式起卦（推荐）
**触发词：** 起卦、算一卦、摇卦、🎲，或直接问问题

**流程（只认笑脸😊）：**
```
用户：今年能不能升职？
系统：💭 您的问题：今年能不能升职
      
      🎲 请发送笑脸表情😊开始摇卦（第1次/共6次）：
      
用户：[😊]
系统：第1次（初爻）：🌕🌕🌑 → 少阳 ⚊
      请发送笑脸表情😊继续（第2次/共6次）：
      
用户：[👍]
系统：❌ 请发送笑脸表情😊继续摇卦
      
用户：[😊]
系统：第2次（二爻）：🌑🌕🌑 → 少阴 ⚋
      请发送笑脸表情😊继续（第3次/共6次）：

用户：OK
系统：【打断】检测到文字输入，已暂停起卦。您想说什么？
```

**输入规则（严格）：**
1. **笑脸表情 😊** → 继续摇卦，每次只处理1步
2. **其他表情**（👍🎉🌕🌑等）→ 提示"❌ 请发送笑脸表情😊继续摇卦"
3. **文字输入**（包括纯文字、混合内容）→ 打断流程，询问意图
1. **一句话触发**（推荐）
   - `今年能不能升职？`
   - `起卦 今年能不能升职`
   - `算一卦 这段感情怎么样`

2. **传统方式**（兼容）
   - `起卦` → 系统回应 → 用户发送问题 → 开始摇卦

**爻的判定规则：**
| 硬币 | 爻名 | 符号 | 动爻 |
|------|------|------|------|
| 🌕🌕🌕 | 老阳 | ⚊ | ✓ |
| 🌑🌑🌑 | 老阴 | ⚋ | ✓ |
| 🌕🌕🌑 | 少阳 | ⚊ | ✗ |
| 🌕🌑🌑 | 少阴 | ⚋ | ✗ |

### 2. 直接输入模式
**触发词：** 六爻、占卦、帮我算一卦（需包含6个爻）

**流程：**
1. 用户直接输入问题+6个爻
2. 系统调用 `liuyao_interpret.py`
3. 用 `message` 发送完整解读结果

## 技术实现

### 输入识别逻辑
```python
if is_emoji(user_input):
    # 纯表情 → 继续摇卦（只处理1次）
    process_one_step()
elif is_text(user_input):
    # 文字输入 → 打断流程
    if has_active_session():
        return "检测到文字输入，已暂停起卦。您想说什么？"
    else:
        return normal_text_processing(user_input)
```

### 核心原则
- 纯表情 = 摇卦信号，继续流程
- 任何文字 = 意图打断，暂停起卦

## 工具调用规范

**所有消息必须使用 `message` 工具发送：**
- action: send
- channel: feishu
- target: 当前会话（自动继承）

**禁止使用：**
- feishu_im_user_message（以用户身份发送，显示错误）

## 状态记录
- 使用 `~/.openclaw/.liuyao_session.json` 记录起卦状态
- 记录内容：用户问题、当前步数(1-6)、已记录的爻
- 超时：5分钟无操作自动重置

## 脚本入口

### 智能处理（推荐）
```bash
python3 liuyao_interactive.py handle "用户输入"
```

### 命令列表
| 命令 | 说明 | 示例 |
|------|------|------|
| `start [问题]` | 开始新起卦 | `python3 liuyao_interactive.py start "今年能不能升职"` |
| `step [输入]` | 进行一次摇卦 | `python3 liuyao_interactive.py step "[表情]"` |
| `status` | 查看当前状态 | `python3 liuyao_interactive.py status` |
| `clear` | 重置状态 | `python3 liuyao_interactive.py clear` |
| `check <输入>` | 测试智能识别 | `python3 liuyao_interactive.py check "今年能不能升职"` |
| `handle <输入>` | 智能处理（推荐） | `python3 liuyao_interactive.py handle "用户输入"` |
| `interpret` | 直接解读 | `python3 liuyao_interpret.py` |
