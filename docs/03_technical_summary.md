## 三大技术难点及解决思路

### 难点1：六十四卦精确匹配

**问题**：用户给出6个爻的结果，需要快速找到对应卦象。

**解决思路**：
- 预建 `hexagrams_full.json`，每个卦象存完整的6位符号串（如 `⚊⚊⚊⚊⚊⚊`）
- 输入解析后，将6个爻转为上爻→初爻的6位符号串
- 用完整符号串做精确匹配查找

**核心代码**：
```python
def get_guaxiang_full(self):
    symbols = [y["symbol"] for y in self.yao_results]  # 初爻在index0
    return "".join(reversed(symbols))  # 上爻在前

def find_hexagram(self):
    guaxiang = self.get_guaxiang_full()
    for key, info in hexagrams.items():
        if info.get("guaxiang_full") == guaxiang:
            return info
```

### 难点2：纳甲的准确装卦

**问题**：纳甲涉及天干地支的复杂对应关系，不同卦宫规则不同。

**解决思路**：
- 先确定卦宫（乾/坤/坎/离/震/巽/艮/兑），根据内外卦查表
- `naga.json` 内置各卦宫的纳甲规则（歌诀数据化）
- 六亲根据"卦宫五行"与"爻地支五行"的关系推算：
  - 卦宫五行生爻五行 → 父母
  - 卦宫五行克爻五行 → 妻财
  - 爻五行与卦宫同 → 兄弟
  - 爻五行生卦宫五行 → 子孙
  - 爻五行克卦宫五行 → 官鬼

**纳甲歌诀（已数据化）**：
| 卦宫 | 内卦（初二三爻） | 外卦（四五上爻） |
|------|------------------|------------------|
| 乾 | 甲起子、寅、辰 | 壬起午、申、戌 |
| 坤 | 乙起未、巳、卯 | 癸起丑、亥、酉 |
| 坎 | 戊起子、寅、辰 | 戊起午、申、戌 |
| ... | ... | ... |

### 难点3：旺衰的月令判断

**问题**：六爻旺衰依赖月令日令，需准确日期和节气信息。

**解决思路**：
- Python `datetime` 获取当前年月
- 年支：`(year - 4) % 12`（已知 1900年是庚子年）
- 月支：固定对应（寅月=正月，卯月=二月...）
- 月令旺相表内置（查表判断五行状态）

**月令旺相表（内置）**：
```
寅卯月：木旺、火相、土死、金囚、水休
巳午月：火旺、土相、金死、水囚、木休
申酉月：金旺、水相、木死、火囚、土休
亥子月：水旺、木相、火死、土囚、金休
辰戌丑未月：土旺、金相、水死、木囚、火休
```

## 能力边界

### ✅ 能做到

| 功能 | 说明 |
|------|------|
| 卦象识别 | 给6个爻→输出完整卦象 |
| 纳甲装卦 | 自动装天干地支六亲 |
| 世应定位 | 标注入卦的世爻应爻 |
| 动变分析 | 识别动爻，生成变卦 |
| 用神选取 | 按问题类型自动选 |
| 格局判断 | 游魂/归魂/三合局等 |
| 旺衰参考 | 月令旺相判断 |
| 吉凶提示 | 综合维度初步判断 |

### ❌ 不能做到

| 功能 | 原因 |
|------|------|
| 精确日令旺衰 | 需要完整农历/节气系统 |
| 流年大运分析 | 没有用户完整八字 |
| 梅花心易 | 梅花重"心易+外应"而非铜钱 |
| 铁板神数 | 需精确出生时辰+复杂算法 |
| 风水调整 | 六爻测事，风水另需专业体系 |
| 医疗诊断 | 可测健康但不能替代医疗 |
