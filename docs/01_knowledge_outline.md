# 六爻知识体系梳理

## 一、起卦方式

### 铜钱摇卦
- 工具：3枚铜钱（或硬币）
- 摇6次，从**初爻**到**上爻**依次记录
- 结果判定：
  - 3正面 → **老阳（⚊ —）** → 变阴
  - 3背面 → **老阴（⚋ ═）** → 变阳
  - 2正1背 → **少阳（⚊ —）** → 静爻
  - 2背1正 → **少阴（⚋ ═）** → 静爻
- **老阳、老阴为动爻**，变卦时阴阳互换

### 爻的表示
```
上爻（第六爻）
五爻（第五爻）
四爻（第四爻）
三爻（第三爻）
二爻（第二爻）
初爻（第一爻）← 从下往上数
```

## 六、动变规则

### 主卦、变卦
- **主卦**（本卦）：摇出的原始卦象
- **变卦**：动爻阴阳互换后的结果
- **动爻**：老阳、老阴所在之爻

### 变爻规则
```
老阳（⚊ —）→ 变阴（⚋ ═）
老阴（⚋ ═）→ 变阳（⚊ —）
少阳少阴不变
```

### 实例
```
主卦：乾为天（⚊⚊⚊⚊⚊⚊）
上爻动 → 变卦：天风姤（⚊⚊⚊⚊⚊⚋）
```

### 月建旺相休囚死
| 月令 | 旺 | 相 | 死 | 囚 | 休 |
|------|----|----|----|----|----|
| 寅卯月 | 木 | 火 | 土 | 金 | 水 |
| 巳午月 | 火 | 土 | 金 | 水 | 木 |
| 申酉月 | 金 | 水 | 木 | 火 | 土 |
| 亥子月 | 水 | 木 | 火 | 土 | 金 |
| 辰戌丑未月 | 土 | 金 | 水 | 木 | 火 |

### 日建作用
- 日建为**当日地支**
- 日建对卦爻的作用：**生克冲合**
- 日建入爻（如爻的地支与日建相同）= 日令扶助

### 动变生克
- 动爻可以生克静爻
- 变爻只能生克动爻
- 静爻不能主动生克他爻

**归魂** → 回归稳定、有归宿感

### 三合局
```
申子辰合水局    寅午戌合火局
亥卯未合木局    巳酉丑合金局
辰戌丑未合土局
```
形成条件：卦中三个爻地支分别为三合局地支，且有一个动爻

### 六冲六合
- **六冲**：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲
- **六合**：子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合

### 经典断语
1. **用神旺相** → 事宜成
2. **用神休囚死** → 事宜缓
3. **世应相生** → 吉；相克 → 凶
4. **动爻克静爻** → 动爻有力
5. **日建入爻** → 日令扶助该爻
6. **月建冲爻** → 月破，该爻无力
7. **三合局成** → 大吉或大凶（看用神）
