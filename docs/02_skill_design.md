# 六爻占卜 Skill 设计方案

## 一、Skill 概述

**名称**：liuyao-skill
**触发词**：六爻、占卦、摇卦、帮我算一卦、起卦
**核心能力**：用户输入问题 + 6个爻的阴阳结果 → 输出完整卦象解读
**目标用户**：航哥（KK AI助手用户）

## 四、输入输出协议

### 输入解析

用户输入示例：
```
我最近财运如何？
初爻：少阳
二爻：少阴
三爻：老阳动
四爻：少阳
五爻：少阴
上爻：少阳
```

解析逻辑：
1. 第一行 = 问题
2. 后续包含"爻"字且含"阳/阴"的行 = 爻信息
3. 格式：`X爻：[少/老][阳/阴][动]`
4. 按位置排序（初爻→上爻）

### 输出协议

最终输出为 Markdown 格式，通过飞书消息发送给用户。

## 五、技术难点与解决思路

### 难点1：六十四卦精确匹配

**问题**：用户给出6个爻的结果，需要快速找到对应卦象。卦象符号组合有 2^6=64 种可能。

**解决思路**：
1. 将6个爻转为6位二进制字符串（上爻为最高位）
2. 作为 key 直接查 `hexagrams.json`
3. 若找不到，用上卦+下卦组合定位

**代码**：
```python
def get_guaxiang_key(yao_results):
    # yao_results 从初爻到上爻
    # 老阳/少阳=1, 老阴/少阴=0
    bits = "".join(["1" if y["yang"] else "0" for y in yao_results])
    return bits  # 如 "000000" = 坤为地
```

### 难点2：纳甲的准确装卦

**问题**：纳甲涉及天干地支的复杂对应关系，不同卦宫规则不同。

**解决思路**：
1. 先确定卦宫（乾/坤/坎/离/震/巽/艮/兑）
2. 根据卦宫确定内卦（初二三爻）和外卦（四五上爻）
3. 按纳甲歌诀给每个爻装上天干地支
4. 六亲根据"卦宫五行"与"爻五行"的关系推算

**简化规则**：
- 乾宫卦：内卦甲壬起子寅辰，外卦甲壬起午申戌
- 其他卦宫类似，查表为主

### 难点3：用"当前时间"做旺衰判断

**问题**：六爻旺衰依赖月令日令，需要准确日期。

**解决思路**：
1. Python `datetime` 获取当前年月日
2. 年支：`(year - 4) % 12` 对应地支表
3. 月支：固定对应（寅月=1月，卯月=2月...）
4. 日支：实际需按"掐指"算法，这里简化为 `(day-1) % 12`
5. 完整版可接入农历转换库（如 `lunarcalendar`）

**月令旺相表**（内置）：
| 月令 | 旺 | 相 | 死 | 囚 | 休 |
|------|----|----|----|----|----|
| 寅卯 | 木 | 火 | 土 | 金 | 水 |
| 巳午 | 火 | 土 | 金 | 水 | 木 |
| 申酉 | 金 | 水 | 木 | 火 | 土 |
| 亥子 | 水 | 木 | 火 | 土 | 金 |
| 辰戌丑未 | 土 | 金 | 水 | 木 | 火 |

## 六、Skill 能力边界

### ✅ 这个 Skill 能做到：

1. **基础卦象解读**：给定6个爻的结果，输出卦名、卦辞、象辞
2. **纳甲装卦**：给每个爻装上天干地支，标注世应位置
3. **用神选取**：根据问题类型自动选取正确的用神
4. **动变分析**：识别动爻，分析变卦
5. **格局判断**：识别游魂卦、归魂卦、三合局等特殊格局
6. **旺衰判断**：基于月令判断用神旺衰（简化版）
7. **吉凶提示**：结合多维度给出初步吉凶结论

### ❌ 这个 Skill 不能做到：

1. **精确日令旺衰**：日令的复杂生克冲合关系（需要完整历法系统）
2. **流年大运分析**：没有用户完整八字，无法做深度命理分析
3. **梅花易数**：梅花侧重"心易"和"外应"，六爻侧重铜钱摇卦
4. **铁板神数**：需精确出生时辰和复杂算法
5. **风水调整建议**：六爻测事，风水另需专业分析
6. **医疗诊断**：虽然可测健康疾病，但不能替代医疗

### ⚠️ 需要注意：

1. **娱乐属性**：六爻是传统占卜文化，本 skill 定位为文化解读工具，非科学预测
2. **信息不完整**：没有用户八字详情，无法做个性化深度分析
3. **时间误差**：月令以节气为准，当前为简化版（以月建为准）
4. **心理暗示**：占卜结果有心理暗示成分，解读需客观
